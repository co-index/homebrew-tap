# homebrew-tap

Homebrew formulae for [co-index](https://github.com/co-index) projects.

## Usage

```sh
brew install co-index/tap/ccnotify
```

## Formulae

| Formula | Description |
| --- | --- |
| [ccnotify](https://github.com/co-index/ccnotify) | Clickable macOS notifications from the command line |
